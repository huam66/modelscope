# AI 应用技术栈（自顶向下）

## 1. LLM 应用（顶层应用）
- RAG 系统
- 智能客服
- 聊天机器人
- 自动化 Agent
- 文档摘要与翻译

## 2. 应用开发框架
- **LangChain**
  - Document Loaders（文档加载器）
  - Text Splitters（文本分割器）
  - Embeddings（嵌入模型接口）
  - Vector Stores（向量存储）
  - Chains（链式调用）
  - Agents（智能体）
  - Memory（对话记忆）

## 3. 模型加载与使用库
- **HuggingFace Transformers**
  - AutoModel / AutoTokenizer
  - from_pretrained 统一接口
  - Pipeline 高级封装
  - 支持的模型：BERT、GPT、Qwen、BGE 等

## 4. 预训练模型
- **嵌入模型（Embedding Models）**
  - BGE-small-zh-v1.5
  - text2vec
  - OpenAI embeddings
- **生成模型（LLMs）**
  - Qwen（千问）
  - GPT 系列
  - LLaMA
  - ChatGLM

## 5. 模型架构
- **Transformer**
  - 自注意力机制（Self-Attention）
  - 多头注意力（Multi-Head Attention）
  - 前馈网络（FFN）
  - 位置编码（Positional Encoding）
  - 残差连接与层归一化
  - 编码器-解码器结构

## 6. 底层深度学习框架
- **PyTorch**
  - 张量计算（Tensor）
  - 自动微分（Autograd）
  - GPU 加速（CUDA）
  - nn.Module 模块化
  - 优化器（Optimizers）
  - 数据加载（DataLoader）
